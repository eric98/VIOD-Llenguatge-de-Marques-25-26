# Exercici 5 - Joc "Pokémon inicial"
Recrea la bafarada de text de quan elegeixes un pokémon.

Referència:
https://static.wikia.nocookie.net/essentialsdocs/images/3/3a/ChoosingStarter.png/revision/latest?cb=20190219205247
